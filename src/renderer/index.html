🔧 Setup Guide

  1. Go to discord.com/developers and create an application
  2. Copy the Client ID and paste it above
  3. Under Rich Presence → Art Assets, upload:
    • ytmusic — YouTube Music logo (large image)
    • playing — Play icon (small image)
    • paused — Pause icon (small image)
  4. Make sure Discord desktop app is running
